# Claude MCP Configuration

This directory contains Model Context Protocol (MCP) server configurations for Claude.

## Figma MCP Server

To use the Figma MCP server:

1. Open `.claude/mcp_config.json`
2. Replace `your-figma-token-here` with your actual Figma Personal Access Token
3. You can get your Figma token from: https://www.figma.com/developers/api#access-tokens

**Important:** Never commit your actual Figma token to the repository. Keep it private and local to your machine.
